## Usage

### CLI Option

```bash
node-janitor --lang vi
```

### Config File

```json
{
  "lang": "vi"
}
```

### Environment Variable

```bash
export NODE_JANITOR_LANG=vi
```

## Contributing Translations

Want to add a new language? Contributions are welcome!

1. Fork the repository
2. Copy `src/utils/i18n.ts`
3. Add your translations
4. Submit a Pull Request

Translation structure:

```typescript
const translations = {
  your_lang: {
    scanning: 'Your translation...',
    found: 'Found {count} folders',
    // ... more keys
  }
};
```
